## Deskripsi Project

Sistem Akademik merupakan aplikasi web sederhana berbasis PHP, MySQL, dan Bootstrap yang dibuat untuk memenuhi tugas Ujian Tengah Semester (UTS) mata kuliah Pemrograman Web II.

Aplikasi ini memiliki fitur autentikasi pengguna (Login dan Register) serta Dashboard yang menyediakan dua modul CRUD (Create, Read, Update, Delete), yaitu:

1. Data Mahasiswa
2. Data Mata Kuliah

Sistem menggunakan database MySQL sebagai media penyimpanan data dan PHP sebagai bahasa pemrograman server-side.

## Fitur Aplikasi

### 1. Register
Pengguna dapat membuat akun baru dengan mengisi:
- Username
- Password
- Status (Admin/User)

### 2. Login
Pengguna dapat masuk ke sistem menggunakan:
- Username
- Password
- Status

### 3. Dashboard
Menampilkan:
- Informasi pengguna
- Menu Data Mahasiswa
- Menu Data Mata Kuliah
- Tombol Logout

### 4. CRUD Data Mahasiswa
- Tambah Data
- Tampilkan Data
- Edit Data
- Hapus Data

### 5. CRUD Data Mata Kuliah
- Tambah Data
- Tampilkan Data
- Edit Data
- Hapus Data

### 6. Logout
Mengakhiri sesi pengguna dan kembali ke halaman login.
